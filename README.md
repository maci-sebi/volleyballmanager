## O projektu

VolleyManager je jednoduchý informační systém určený pro trenéry, manažery a hráče volejbalových týmů. Cílem je zjednodušit plánování tréninků a zápasů, evidovat docházku, spravovat hráče a ukládat základní statistiky.

## Funkce

- Plánování tréninků a zápasů
- Evidence docházky (přítomen / omluven / neomluven)
- Správa hráčů (profil, kontakty, pozice)
- Tvorba sestav pro zápasy
- Zadávání zápasových statistik a poznámek
- Odesílání oznámení hráčům

## Role a oprávnění

- **Trenér (Admin)**: plná správa hráčů, plánování, statistiky, zprávy
- **Hráč**: zobrazení profilu, potvrzení účasti, přístup ke svým statistikám
- **Statistik**: tvoření událostí, tréninků, plánování, zprávy

## Spravovaná data

- Hráči: jméno, věk, pozice, kontakty, statistiky, docházka
- Tréninky: datum, čas, místo, obsah, účast
- Zápasy: soupeř, místo, sestava, výsledek, statistiky
- Docházka: přítomen / omluven / neomluven
- Oznámení: týmové zprávy od trenéra
